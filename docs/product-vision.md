# Quick Code Test Prep

## Product Vision (v1)

---

## 1. Overview

Quick Code Test Prep helps people prepare quickly for interview code tests in SQL and Python.

In under 2 hours, users can:

* Refresh core concepts
* Practice common test patterns
* Simulate a timed code screen
* Walk into interviews feeling prepared instead of scrambling

We don’t promise users that they will pass every test.
We help them show up ready.

---

## 2. Core Transformation

Before:

* “I haven’t written SQL in months.”
* “What if they ask me something basic and I blank?”
* “I should review that whole course… but I don’t have time.”

After:

* “I reviewed the key concepts.”
* “I practiced realistic problems.”
* “I won’t freeze if they ask about JOINs.”
* “I feel ready.”

The primary transformation is:
**From rusty and anxious → prepared and steady.**

---

## 3. Target User

### Primary User

Smart professionals who:

* Have learned SQL or Python before
* May not use them daily
* Have an upcoming code test (often in days)
* Feel anxious about being rusty

Examples:

* Technical writers
* Data analysts
* PMs
* Junior developers
* Career switchers
* Bootcamp grads
* Adjacent-to-technical professionals

### Secondary User

True beginners who need:

* A fast crash course
* Interview-relevant foundations
* Not a full-length curriculum

---

## 4. Problem Statement

Existing options are poorly matched to this situation:

* Full-length courses are too long.
* LeetCode-style platforms are too advanced.
* Reviewing old course material is inefficient.
* General chat tools lack structure and accountability.

There is a gap between:
“Learn programming”
and
“Pass a code screen tomorrow.”

Quick Code Test Prep fills that gap.

---

## 5. v1 Scope (Locked)

v1 includes:

### Skills

* SQL
* Python

### Per Skill

1. 30-Minute Crash Course
2. Warmup Tests (5 short drills)
3. Timed Test (20-minute simulation)
4. Structured AI feedback
5. Basic progress tracking

v1 does NOT include:

* Resume upload
* Job description parsing
* Adaptive curriculum trees
* Company-specific simulations
* Gamification
* Community features
* Advanced personalization

Scope discipline is critical for 8–12 week launch.

---

## 6. Product Structure (v1)

### Public Pages

For each skill (SQL / Python):

Users can view:

* Crash course lesson titles
* Description of Warmup tests
* Description of Timed test

All content descriptions are public.

Actual exercises require:

* Account creation
* 7-day free trial

---

### Inside the App

Per skill:

#### 1. 30-Minute Crash Course

A short, focused set of lessons covering:

SQL:

* SELECT + WHERE
* GROUP BY + Aggregation
* JOINs
* Subqueries + NULL handling

Python:

* Lists + Loops
* Dicts + Conditionals
* Functions + Transformations
* JSON + Basic parsing

Each lesson includes:

* Short explanation
* 1–2 guided examples
* 1 mini exercise
* Immediate structured feedback

---

#### 2. Warmup Tests

* 5 short drills
* Immediate feedback per question
* Designed to “activate” knowledge quickly

Tone:
Encouraging, practical, calm.

---

#### 3. Timed Test

* 20-minute timer
* Realistic scenario
* 2–3 related questions
* Feedback at end

Designed to simulate pressure without overwhelming users.

---

## 7. Tone & Voice

Voice is:

* Calm
* Practical
* Conversational
* Slightly warm
* Never corporate
* Never sarcastic
* Not overly jokey

We acknowledge anxiety without amplifying it.

Example tone:

“You’ve got an interview coming up. Let’s make sure you don’t walk in cold.”

---

## 8. Monetization (v1)

* 7-day free trial
* $29/month subscription
* Cancel anytime

Primary use case:
Short-term interview prep cycles.

Retention does not need to be long-term if value is clear.

## 10. Success Criteria (Early)

Within first 60–90 days post-launch:

* Users complete crash course
* Users complete warmup
* Users return for second session
* At least 10 strong testimonials
* At least 20–50 paying subscribers

Qualitative success indicator:
Users report feeling more confident going into interviews.

---

## 11. What We Are Not

We are not:

* A full coding bootcamp
* A long-form programming course
* A LeetCode clone
* A general-purpose AI tutor

We are:
Targeted, tactical interview readiness.
